# Material Symbols Rounded

`material-symbols-rounded.woff2` is a self-hosted subset of Google's Material
Symbols Rounded variable font. It was downloaded from the official Google Fonts
CSS API and contains the ligatures used by Form Swarm:

`auto_awesome`, `chevron_left`, `chevron_right`, `circle`, `expand_more`,
`favorite`, `link`, `search`, `send`, `star`, and `thumb_up`.

Source: https://fonts.google.com/icons?icon.style=Rounded

CSS API subset: `icon_names=auto_awesome,chevron_left,chevron_right,circle,expand_more,favorite,link,search,send,star,thumb_up`

Upstream: https://github.com/google/material-design-icons
